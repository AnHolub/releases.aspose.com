## Por que baixar o Aspose.OCR para Java?

Aspose.OCR para Java é uma API de reconhecimento de caracteres e marcas ópticas que permite aos desenvolvedores adicionar a funcionalidade OCR em seus aplicativos. Ele fornece um conjunto simples de classes para controlar o reconhecimento de caracteres.

## Converter imagem em texto via Java

- Leia caracteres de imagens.
- Suporte para formatos de arquivo de imagem JPG, PNG, GIF, BMP e TIFF para OCR.
- Suporte para inglês, francês, espanhol e português.
- Leia fontes populares, incluindo Arial, Times New Roman, Courier New, Verdana, Tahoma e Calibri.
- Suporte para estilos de fonte regulares, negrito e itálico.
- Digitalize toda a imagem ou qualquer parte da imagem.
- Digitalize imagens giradas.
- Pode aplicar diferentes filtros de remoção de ruído antes do reconhecimento da imagem.
- Extraia elementos ópticos marcados de imagens digitalizadas.
- Suporte para vários elementos ópticos marcados, como Check Box, Choice Box, Grid, Barcode e OCR.

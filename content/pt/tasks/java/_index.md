## Por que baixar o Aspose.Tasks para Java?

Aspose.Tasks uma API Java Project Management para manipular arquivos do Microsoft Project®. Os desenvolvedores podem definir as propriedades principais e padrão de um projeto, bem como as informações do calendário. A API também oferece a capacidade de especificar exceções de dias da semana e calendário, ao mesmo tempo que facilita a adição e o recalculo de tarefas, recursos e outros dados sem intervenção do usuário.

## Processamento de arquivos do Microsoft Project®

- Ler, alterar e escrever documentos do Microsoft Project®.
- Leia formatos de projeto MPP e XML.
- Criar, atualizar e escrever projetos em formato de projeto XML.
- Leia arquivos de projeto MPP, atualize informações resumidas e escreva no formato MPP original.
- Leia arquivos de projeto MPP e converta-os para o formato XML.
- Altere as principais configurações do projeto, como tipo de cronograma, datas de início e término.
- Altere as configurações padrão do projeto, como taxa padrão padrão, taxa padrão de horas extras, tipo de tarefa padrão, acúmulo de custo fixo padrão, etc.
- Gerenciar atributos estendidos.
- Definir dias da semana para o projeto, bem como para calendários e exceções de calendário.
- Ler e escrever calendários para tarefas e recursos.
- Gerenciar agendamento e duração da linha de base da tarefa.
- Lidar com restrições em tarefas.
- Criar e gerenciar links entre tarefas.
- Ler, alterar e criar tarefas, marcos, tarefas críticas estimadas ou orientadas por esforço.
- Gerenciar custos e variações de recursos.
- Custos de atribuição de acesso e orçamento.
- Implementação de prefixo de recurso para recursos aninhados

### Formatos de projeto de leitura e gravação

MPP, XML, MPT, MPX

### Salvar arquivos de projeto como

XLSX, HTML, TXT, PDF, XER, PrimaveraP6XML\
TIF, SVG, PNG, JPEG

- **macOS:** Mac OS X
- **Linux:** Ubuntu, OpenSUSE, CentOS e outros
- **Versões Java:** `J2SE 7.0 (1.7)` ou superior

## Por que baixar o Aspose.BarCode para Java?

[Aspose.BarCode for Java](https://products.aspose.com/barcode/java) é um componente de geração e reconhecimento de código de barras robusto e confiável, escrito em Java, que permite que os desenvolvedores adicionem de forma rápida e fácil a funcionalidade de geração e reconhecimento de código de barras aos seus aplicativos Java.

## Recursos da API de código de barras

- Especifique diferentes tipos de simbologias.
- Definir o texto do código de barras, aparência e outras propriedades.
- Suporte de Checksum e Suplemento de Dados.
- Imprima imagens de código de barras.

### Simbologias de código de barras

- **Somente numérico:** EAN13, EAN8, UPCA, UPCE, BooklandEAN, Interleaved2of5, Standard2of5, MSI, Code11, Codabar, Postnet, Planet, EAN14(SCC14), SSCC18, ITF14, Leticode, OPC
- **Alfa-Numérico:** Code128, EAN128, Code39 Extended, Code39 Standard, Code93, Extended, Code93 Standard, Matrix 2 of 5, PZN, Deutsche Post Identcode, VIN
- **Simbologias 2D:** PDF417, DataMatrix, QR, Swiss QR (QR Bill)
- **Suporte somente para codificação:** Australia Post, Aztec

### Ler e escrever etiquetas de código de barras

**Imagens:** JPEG, TIFF, PNG, BMP, GIF, EXIF, EMF, SVG

### Salvar códigos de barras como

**Imagens:** JPEG, TIFF, PNG, BMP, GIF, EXIF, EMF, SVG

### Ambientes Suportados

- **Qualquer sistema operacional compatível com Java:**
- **Tempo de execução Java:** `JRE 1.8` ou superior
- **Desenvolvimento Java:** `J2SE 8.0 (1.8)`, ou superior.

# Mesclagem de cenas 3D simplificada

Simplificou as funções para mesclar cenas. Este novo método permite mesclar tudo de outro nó para o nó atual.

```csharp
var scene1 = Scene.FromFile("file1");
var scene2 = Scene.FromFile("file2");
scene1.RootNode.Merge(scene2.RootNode);
scene1.Save("output.fbx", FileFormat.FBX7700Binary);
```

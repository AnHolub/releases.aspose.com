É uma API de imagem independente que consiste em rotinas Java que permitem que seus aplicativos Java desenhem, bem como executem processamento de nível básico a avançado de imagens raster e vetoriais.

## Por que baixar o Aspose.Imaging para Java?

O Aspose.Imaging para Java oferece compactação de imagem robusta e alta velocidade de processamento por meio de acesso a byte nativo e uma variedade de algoritmos eficientes. Ele não apenas manipula, exporta e converte imagens, mas também permite desenhar objetos dinamicamente usando manipulação de pixels e caminho gráfico.

## Recursos da API de imagem

- Desenhe imagens raster com gráficos.
- Desenhe imagens vetoriais.
- Conversão de imagens para vários formatos.
- Aplique mascaramento, bem como filtros Median e Wiener.
- Corte, gire e redimensione imagens via API.
- Desinclinar e transformar imagens.
- Definir propriedades da imagem.

### Ler e gravar formatos de imagem

**Formatos de varredura:** JPEG2000, JPEG, BMP, TIFF, GIF, PNG, APNG\
**Metarquivos:** EMF, EMZ, WMF, WMZ\
**Outros:** SVG, SVGZ, DICOM

### Salvar imagens como

**Corrigido:** PDF\
**Photoshop:** PSD\
**Marcação:** Tela HTML5

### Ler formatos de imagem

**Vários:** DjVu, DNG, ODG, CMX, CDR, DIB, OTG, FODG, EPS (somente visualização de raster), WEBP

### Sistemas operacionais suportados

O Aspose.Imaging for Java pode ser executado virtualmente em qualquer sistema operacional onde o Java esteja instalado (desde o JDK 1.6)

- Windows (desde 7)
- Linux
- Mac OS
- Qualquer SO onde o Java esteja instalado.

O Aspose.Imaging funciona para as versões x86 e x64 dos sistemas operacionais listados acima.

**Nota:** No sistema operacional Linux, é recomendado instalar o pacote com fontes compatíveis com Microsoft (por exemplo, `sudo apt-get install ttf-mscorefonts-installer`).

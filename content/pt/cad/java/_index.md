## Por que baixar o Aspose.CAD para Java?

O Aspose.CAD para Java permite que os desenvolvedores convertam arquivos AutoCAD® DWG e DXF em imagens PDF e Raster. É uma API nativa e não requer a instalação do AutoCAD® ou de qualquer outro software. Você também pode converter as camadas e layouts selecionados dos arquivos do AutoCAD®. A conversão para imagens PDF e Raster é de altíssima qualidade.

## Processamento de arquivos CAD

- Ajuste o tamanho do desenho CAD.
- Converta desenhos CAD para outros formatos de arquivo.
- Exporte imagens 3D AutoCAD® para PDF.
- Exportação de layouts CAD para PDF.
- Defina o tempo limite ao salvar para evitar gastar muito tempo ou consumir muita memória.

### Ler Formatos CAD

**AutoCAD®:** DWG, DWT, DWF, DWXF, IFC, PLT\
**MicroStation:** DGN\
**O Visualizador Avançado:** OBJ\
**Outros:** STL, IGES, CFF2

### Salvar CAD como

**Layout fixo:** PDF\
**Imagens rasterizadas:** PNG, BMP, TIFF, JPEG, GIF

### Ler escrever

**CAD:** DXF\
(Recursos de gravação são parcialmente suportados.)

### Ambientes Suportados

- **Microsoft Windows:** Windows Desktop & Server (x86, x64)
- **macOS:** Mac OS X
- **Linux:** Ubuntu, OpenSUSE, CentOS e outros
- **Versões Java:** `J2SE 7.0 (1.7)`, `J2SE 8.0 (1.8)`

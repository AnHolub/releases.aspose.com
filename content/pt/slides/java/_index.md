## Por que baixar Aspose.Slides para Java?

[Aspose.Slides](https://products.aspose.com/slides/java) para Java é uma API local para trabalhar com as apresentações do Microsoft PowerPoint® em seus aplicativos Java. Ele permite que os desenvolvedores leiam, escrevam, convertam e manipulem apresentações do PowerPoint® em aplicativos Java com a capacidade de manipular vários elementos de documentos, como slides, tabelas, texto, gráficos, formas, imagens e diagramas SmartArt e muito mais.

## Manipulação de arquivos do PowerPoint® via Java

- Carregue e converta apresentações para outros formatos.
- Crie apresentações do zero.
- Manipule todos os elementos de apresentação via modelo de objeto intuitivo.
- Criar ou incorporar gráficos.
- Criar ou manipular formas.

Verifique o [product overview](https://docs.aspose.com/slides/java/product-overview/) para saber tudo sobre o Aspose.Slides para Java.

### Ler e escrever apresentações

**Microsoft PowerPoint®:** PPT, PPTX, PPS, POT, PPSX, PPTM, PPSM, POTX, POTM\
**OpenOffice:** ODP, OTP

### Salvar apresentações como

**Layout fixo:** PDF, PDF/A, XPS\
**Imagem:** JPEG, PNG, BMP, TIFF, GIF, SVG\
**Web:** HTML, SWF

### Ambientes Suportados

- **Microsoft Windows®:** Windows Desktop & Server (x86, x64)
- **macOS:** Mac OS X
- **Linux:** Ubuntu, OpenSUSE, CentOS e outros
- **Versões Java:** `J2SE 6.0 (1.6)` ou superior

## Por que baixar o Aspose.Words para Java?

[Aspose.Words for Java é uma API avançada de processamento de texto Java que permite que você execute uma grande variedade de tarefas de processamento de documentos diretamente em seus aplicativos Java. Aspose.Words para API Java suporta processamento de palavras (DOC, DOCX, OOXML, RTF) HTML, OpenDocument, PDF, EPUB, XPS, SWF e todos os formatos de imagem. Com Aspose.Words você pode gerar, modificar e converter documentos sem usar o Microsoft Word®.

## Recursos da API do Word

### Renderização e impressão

- Layout do documento em páginas com alta fidelidade (exatamente como o Microsoft Word® faria) para todos os formatos abaixo.
- Renderize páginas individuais ou documentos completos para `PDF`, `XPS` ou `SWF`.
- Renderizar páginas de documentos para imagens raster (Multipage `TIFF`, `PNG`, `JPEG`, `BMP`).
- Renderizar páginas para um objeto Java Graphics para um tamanho específico.
- Imprima páginas de documentos usando a infraestrutura de impressão Java.
- Atualize o sumário, números de página e outros campos antes de renderizar ou imprimir.
- Renderização de Efeitos 3D através do `OpenGL`.

### Recursos de conteúdo do documento

- Acesse, crie e modifique vários elementos do documento.
- Acesse e modifique todos os elementos do documento usando classes e métodos do tipo `XmlDocument`.
- Copie e mova elementos do documento entre documentos.
- Junte e divida documentos.
- Especifique a proteção de documentos, abra documentos protegidos e criptografados.
- Localizar e substituir texto, enumerar sobre o conteúdo do documento.
- Preservar ou extrair objetos OLE e controles ActiveX do documento.
- Preservar ou remover macros VBA do documento. Preservar a assinatura digital de macros VBA.

### Recursos de relatórios

- Suporte de sintaxe C# e métodos de extensão LINQ diretamente em templates (mesmo para fontes de dados `ADO.NET`).
- Suporte de blocos de documentos repetíveis e condicionais (loops e condições) para tabelas, listas e conteúdo comum.
- Suporte de gráficos e imagens gerados dinamicamente.
- Suporte de inserção de documentos externos e blocos `HTML` em um documento.
- Suporte de múltiplas fontes de dados (inclusive de diferentes tipos) para a geração de um único documento.
- Suporte embutido de relações de dados (master-detail).
- Suporte abrangente de várias manipulações de dados, como agrupamento, classificação, filtragem e outros diretamente em modelos.

Para obter uma lista mais abrangente de recursos, visite [Feature Overview](https://docs.aspose.com/words/java/feature-overview/).

### Ler e escrever formatos de documentos

**Microsoft Word®:** DOC, DOCX, RTF, DOT, DOTX, DOTM, DOCM FlatOPC, FlatOpcMacroEnabled, FlatOpcTemplate, FlatOpcTemplateMacroEnabled\
**OpenOffice®:** ODT, OTT\
**WordprocessingML:** WordML\
**Web:** HTML, MHTML\
**Layout fixo:** PDF\
**Texto:** TXT\
**Outro:** MD

### Salvar arquivos do Word como

**Layout fixo:** XPS, OpenXPS, PostScript (PS)\
**Imagens:** TIFF, JPEG, PNG, BMP, SVG, EMF, GIF\
**Web:** HtmlFixed\
**Outros:** PCL, EPUB, XamlFixed, XamlFlow, XamlFlowPack

### Ler formatos de arquivo

**MS Office®:** DocPreWord60\
**e-book:** MOBI

### Ambientes Suportados

- **Microsoft Windows®:** Windows Desktop & Server (x86, x64)
- **macOS:** Mac OS X
- **Linux:** Ubuntu, OpenSUSE, CentOS e outros
- **Versões Java:** `J2SE 7.0 (1.7)`, `J2SE 8.0 (1.8)` ou superior.

## Γιατί να κατεβάσετε το Aspose.PUB για Java;

Το Aspose.PUB for Java είναι ένα Java API που μπορεί να διαβάζει, να χειρίζεται και να μετατρέπει αρχεία Microsoft Publisher® (.pub) σε PDF. Μπορείτε να χρησιμοποιήσετε το API στις εφαρμογές σας Java για να εργαστείτε με αρχεία .pub χωρίς να ανησυχείτε για την υποκείμενη μορφή αρχείου. Το API μπορεί να χρησιμοποιηθεί σε επιτραπέζιους υπολογιστές Java καθώς και σε διαδικτυακές εφαρμογές χρησιμοποιώντας JSP. Το API είναι ελαφρύ, εύκολο στη χρήση και απλοποιεί την εφαρμογή σας με απλές γραμμές κώδικα στην εφαρμογή σας.

## Δυνατότητες PUB Java API

- Διαβάστε αρχεία Microsoft Publisher® (PUB) για μετατροπή σε μορφή PDF.
- Εργασία με Μεταδεδομένα αρχείων PUB.

### Διαβάστε τα αρχεία PUB

**Microsoft Publisher®:** PUB

### Αποθήκευση PUB ως

**Σταθερή διάταξη:** PDF

### Υποστηριζόμενα περιβάλλοντα

- **Microsoft Windows®:** Windows Desktop & Server (x86, x64)
- **macOS:** Mac OS X
- **Linux:** Ubuntu, CentOS και άλλα
- **Εκδόσεις Java: ** `J2SE 8.0 (1.8)` ή νεότερη έκδοση

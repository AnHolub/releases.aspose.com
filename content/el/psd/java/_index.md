## Γιατί να κατεβάσετε το Aspose.PSD για Java;

Το Aspose.PSD για Java είναι ένα εύχρηστο API χειρισμού μορφής αρχείου Adobe Photoshop®. Μπορεί εύκολα να φορτώσει και να διαβάσει αρχεία PSD, PSB και AI, δίνοντας τη δυνατότητα στους προγραμματιστές Java να εκτελούν λειτουργίες όπως ενημέρωση ιδιοτήτων επιπέδου, προσθήκη υδατογραφημάτων, συμπίεση, περιστροφή, κλιμάκωση ή απόδοση μιας μορφής αρχείου στην άλλη χωρίς να χρειάζεται να εγκαταστήσουν το Adobe Photoshop® .

## Επεξεργασία αρχείων Photoshop®

- Φορτώστε, δημιουργήστε και χειριστείτε αρχεία PSD & PSB.
- Συμπίεση PSD με RLE.
- Περιστροφή, αναστροφή, κλίμακα ή περικοπή εικόνων.
- Ανάγνωση και εγγραφή δεδομένων EXIF.
- Μετατροπή διάνυσμα σε ράστερ.
- Ρύθμιση φωτεινότητας, αντίθεσης και γάμα.
- Σχεδιάστε και συμπληρώστε βασικά σχήματα.
- Κόψτε ορθογώνιες περιοχές.
- Εφαρμόστε το dithering.

### Φορτώστε αρχεία Photoshop® & Illustrator®

**Adobe®:** PSD, PSB, AI

### Αποθήκευση αρχείων Photoshop® & Illustrator® ως

**Raster:** TIFF, JPEG, PNG, GIF, BMP, JPEG2000\
**Σταθερή διάταξη:** PDF

### Υποστηριζόμενα περιβάλλοντα

- **Microsoft Windows®:** Windows Desktop & Server (x86, x64)
- **macOS:** Mac OS X
- **Linux:** Ubuntu, OpenSUSE, CentOS και άλλα
- **Εκδόσεις Java: ** `J2SE 6.0 (1.6)` ή νεότερη έκδοση

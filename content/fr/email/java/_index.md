Aspose.Email pour Java est un ensemble complet d'API de traitement des e-mails pour créer, lire et manipuler des e-mails à partir de vos applications. Il facilite le travail avec de nombreux formats de messages électroniques Outlook tels que les fichiers MSG, EML, EMLX et MHT sans avoir besoin d'installer Microsoft Outlook. Il vous permet également de gérer les fichiers de stockage de messages - Fichiers de stockage personnels (PST), Fichiers de stockage hors ligne (OST) ainsi que les capacités d'envoi et de réception de messages. Vous pouvez également lire et extraire le fichier Outlook PST qui peut être enregistré sur le disque au format MSG.

### Fonctionnalités de l'API de messagerie

- Créez des messages à partir de zéro ou chargez des fichiers de courrier électronique existants pour les modifier.
- Créer et définir le contenu des messages MIME.
- Extraire le contenu des e-mails.
- Charger et enregistrer le rendez-vous au format ICS.
- Possibilité de se connecter au serveur SMTP, POP3, IMAP, Exchange.
- Fonctionne avec Thunderbird, Zimbra et IBM Notes.

### Lire et écrire des formats d'e-mail

**Microsoft Outlook® :** MSG, PST, OST, OFT\
**E-mail :** EML, EMLX, MBOX\
**Autres :** ICS, VCF, HTML, MHTML

### Lire les formats d'e-mail

**Mac Outlook :** OLM

### Environnements pris en charge

- **Microsoft Windows® :** Bureau et serveur Windows (x86, x64)
- **macOS :** Mac OS X
- **Linux :** Ubuntu, OpenSUSE, CentOS et autres
- **Versions Java :** `J2SE 7.0 (1.7)`, `J2SE 8.0 (1.8)`

### Commencer

Aspose héberge toutes les API Java sur [Aspose Repository](https://repository.aspose.com/webapp/#/artifacts/browse/tree/General/repo/com/aspose/aspose-email). Vous pouvez facilement utiliser l'API Aspose.BarCode for Java directement dans vos projets Maven avec des configurations simples. Pour les instructions détaillées, veuillez visiter la page de documentation [Installing Aspose.Email for Java from Maven Repository](https://docs.aspose.com/email/java/installation/).

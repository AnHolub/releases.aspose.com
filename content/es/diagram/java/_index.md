## ¿Por qué descargar Aspose.Diagram para Java?

Aspose.Diagram for Java API es una solución para los requisitos de manipulación de archivos de Microsoft Visio®. Permite que las aplicaciones de Java lean, escriban, exporten y procesen diagramas de Microsoft Visio® mientras admiten todos los objetos y propiedades de Visio®, incluidas formas, páginas, imágenes, patrones de formas, plantillas, texto, capas, encabezados, pies de página, celdas definidas por el usuario, hipervínculos, protección de archivos, geometrías, cuadros de texto y comentarios.

Aspose.Diagram for Java también se puede utilizar para aplicar protección a los dibujos de Microsoft Visio® mediante el bloqueo de fondos, plantillas (maestro), así como formas y estilos para evitar modificaciones accidentales.

## Funciones de procesamiento de archivos de Visio®

- Cargue, cree, guarde y convierta diagramas de Microsoft Visio®.
- Crear Master desde sctrach o agregar desde Stencil of Shapes.
- Cree, distribuya y ajuste automáticamente las formas, así como recupere los conectores de Visio® y la información de fuentes.
- Modificar la sección de geometría de una forma de conector.
- Insertar un Control ActiveX en el Diagrama de Visio®.
- Imprimir en una impresora específica o configurar la impresora y el nombre del documento.

### Leer y escribir formatos de Visio®

**Microsoft Visio®:** VSDX, VSX, VTX, VDX, VSSX, VSTX, VSDM, VSSM, VSTM

### Guardar diagramas de Visio® como

**Diseño fijo:** PDF, XPS\
**Imágenes:** JPEG, PNG, BMP, TIFF, SVG, EMF\
**Web:** HTML\
**Otro:** XAML, SWF

### Leer formatos de Visio®

**Microsoft Visio®:** VDW, VSD, VSS, VST

### Entornos compatibles

- **Microsoft Windows®:** Escritorio y servidor de Windows (x86, x64)
- **macOS:** Mac OS X
- **Linux:** Ubuntu, OpenSUSE, CentOS y otros
- **Versiones de Java:** `J2SE 7.0 (1.7)`, `J2SE 8.0 (1.8)`

Aspose.Total para SharePoint contiene los siguientes componentes:

* Aspose.Words para SharePoint: trabaje con informes, conversión de documentos y flujos de trabajo dentro de SharePoint®.
* Aspose.PDF para SharePoint: exporte listas, elementos de lista y páginas Wiki de SharePoint® al formato de archivo PDF.
* Aspose.Cells for SharePoint: Previsualice, procese y convierta documentos de Microsoft Excel® en SharePoint®.
* Aspose.Email para SharePoint: agregue funciones de conversión de correo electrónico, sincronización de biblioteca de documentos y sincronización de correo electrónico a SharePoint.
* Aspose.Slides for SharePoint: Convierta presentaciones de PowerPoint® a otros formatos admitidos dentro de sus sitios de Microsoft SharePoint®.
* Aspose.Imaging para SharePoint: convierta, edite, cambie el tamaño, rote y voltee imágenes en SharePoint®.
* Aspose.BarCode for SharePoint: Genere códigos de barras postales y 2D en SharePoint®.

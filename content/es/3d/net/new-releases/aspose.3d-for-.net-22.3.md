# Malla grande mejorada en exportación U3D/PDF

Se mejoró la malla grande en la exportación de archivos U3D / PDF.

# Fusión de escena 3D simplificada

Simplificó las funciones para fusionar escenas. Este nuevo método permite fusionar todo, desde otro nodo hasta el nodo actual.

```csharp
var scene1 = Scene.FromFile("file1");
var scene2 = Scene.FromFile("file2");
scene1.RootNode.Merge(scene2.RootNode);
scene1.Save("output.fbx", FileFormat.FBX7700Binary);
```

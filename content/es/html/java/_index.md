La API HTML de Java ayuda a los desarrolladores a escribir, leer, modificar, navegar y convertir documentos (X)HTML desde aplicaciones Java.

## ¿Por qué descargar Aspose.HTML para Java?

Aspose.HTML for Java API funciona como un navegador sin cabeza que le permite crear o abrir documentos HTML existentes de varias fuentes para realizar operaciones de manipulación como eliminar y reemplazar nodos HTML, guardar documentos HTML, extraer CSS de HTML, configurar un documento caja de arena y más. Puede navegar por documentos HTML utilizando varios métodos, como recorrido de elementos, recorrido de documentos, consultas XPath y consultas de selector CSS, así como manipular HTML DOM a través de JavaScript, convertir archivos HTML en imágenes o formatos de diseño fijos, y convertir XHTML y EPUB. archivos a otros formatos de archivo.

## Funciones de procesamiento de HTML

- Escrito completamente en Java y funciona con JRE.
- Soporta tanto el sistema operativo `32-bit` como el `64-bit`.
- Crear o abrir un documento HTML existente de diferentes fuentes.
- Capacidad para manipular (crear, editar, eliminar, reemplazar) nodos HTML a través de API.
- Extrae estilos CSS para un nodo HTML específico.
- Configurar un sandbox de documentos que afecta el procesamiento de documentos HTML.
- Admite la navegación a través de documentos HTML de varias maneras (Element Traversal, Document Traversal, consultas XPath, consultas CSS Selector).
- Manipular HTML DOM a través de JavaScript.
- Convierta documentos web a varios formatos de archivo compatibles.
- Extraer texto de las páginas.
- Buscar texto de páginas.
- Agregar texto en archivo HTML.
- Crear, editar, eliminar y reemplazar nodos HTML
- Extracción de estilos CSS para un nodo HTML particular
- Convierta documentos HTML en varios formatos de imagen admitidos: JPEG, PNG, BMP, TIFF
- Convertir documentos HTML a formato PDF
- Convierte documentos HTML a formato XPS.

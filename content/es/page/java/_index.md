### ¿Por qué descargar Aspose.Page para Java?

Aspose.Page para Java ofrece la creación y manipulación de documentos XPS. También proporciona funcionalidad para convertir documentos XPS, PS y EPS a formatos de imagen y PDF.

### Manipular documentos XPS a través de Java

- Crear y editar documento XPS.
- Agregar o eliminar páginas del documento XPS.
- Guarde documentos XPS como imágenes rasterizadas y PDF.
- Trabajar con lienzos, caminos y elementos de glifos.
- Cree formas de gráficos vectoriales (elemento Path) utilizando un conjunto de primitivas (arcos elípticos, segmentos de curva Bezier y segmentos de línea recta).
- Crear cadenas de texto (elemento Glyphs).
- Elementos de grupo (elemento Canvas) para manipular un grupo como un todo.
- Manipular la apariencia de gráficos y cadenas de texto.
- Use pinceles de diferentes tipos, incluido el pincel de color sólido, el pincel de imagen, el pincel visual y más.
- Especifique colores en diferentes espacios de color, incluidos sRGB, scRGB y cualquier espacio basado en el perfil ICC.
- Trabajar con múltiples documentos dentro de un documento XPS.
- Conservar los boletos de impresión en el documento XPS abierto, si corresponde.
- Agregue boletos de impresión predeterminados en un nuevo documento XPS o en uno abierto si no hay boletos en él.
- Manipular hipervínculos asociados con elementos XPS.
- Manipular, guardar y convertir elementos de esquema XPS.

### Convertir archivos PostScript a través de Java

- Admite los niveles de lenguaje PostScript 1-3 con la excepción de los tipos de fuente: Type2 (CFF), Type14 (Chameleon), Types 9, 10, 11, 32 (CID-Keyed)
- Guarde PS y EPS como PDF e imágenes.

### Entornos compatibles

- **Microsoft Windows®:** Escritorio y servidor de Windows (x86, x64)
- **macOS:** Mac OS X
- **Linux:** Ubuntu, OpenSUSE, CentOS y otros
- **Versiones de Java:** `J2SE 8.0 (1.8)` o superior

## Warum Aspose.OMR für Java herunterladen?

Aspose.OMR für Java ist eine Java-Klassenbibliothek, die eine API bereitstellt, um optische Markierungen aus digitalisierten OMR-Blattbildern zu erkennen. Es kann verwendet werden, um optische Markierungen in einer Vielzahl von Bildformaten wie BMP, JPG, TIF, TIFF, GIF zu erkennen. Die API ermöglicht die Erfassung von von Menschen markierten Daten aus Dokumentformularen wie Umfragen, Fragebögen, Multiple-Choice-Prüfungsunterlagen und anderen Tests. Mit dieser Lösung ist es möglich, gescannte Bilder und sogar Fotos mit hoher Genauigkeit zu erkennen. Die Erkennung basiert auf einem Vorlagen-Markup, das eine grafische Abbildung der zu erkennenden Elemente aus den gescannten Bildern enthält.

## OMR-API-Funktionen

- Erkennung von gescannten Bildern und Fotos.
- Fähigkeit, gedrehte und perspektivische (Seitenansicht) Bilder zu verarbeiten.
- Daten aus Tests, Prüfungen, Fragebögen, Umfragen etc. erkennen
- Hohe Genauigkeitsrate und Möglichkeit, die Ergebnisse im Dateiformat "CSV" und "JSON" zu exportieren.
- Erstellen Sie OMR-Vorlagen aus Textmarkup.

### OME-Ergebnisse speichern unter

### Bilder für OMR lesen

JPEG, PNG, GIF, TIFF, BMP

Aspose.Email für Java ist ein vollständiger Satz von E-Mail-Verarbeitungs-APIs zum Erstellen, Lesen und Bearbeiten von E-Mails aus Ihren Anwendungen heraus. Es erleichtert die Arbeit mit vielen Outlook-E-Mail-Nachrichtenformaten wie MSG-, EML-, EMLX- und MHT-Dateien, ohne dass Microsoft Outlook installiert werden muss. Es ermöglicht Ihnen auch, Nachrichtenspeicherdateien zu verwalten - Personal Storage Files (PST), Offline Storage Files (OST) sowie Funktionen zum Senden und Empfangen von Nachrichten. Sie können auch Outlook PST-Dateien lesen und extrahieren, die im MSG-Format auf der Festplatte gespeichert werden können.

### E-Mail-API-Funktionen

- Erstellen Sie Nachrichten von Grund auf neu oder laden Sie vorhandene E-Mail-Dateien zur Bearbeitung.
- Erstellen und Inhalt von MIME-Nachrichten festlegen.
- Inhalte aus E-Mails extrahieren.
- Termin im ICS-Format laden und speichern.
- Fähigkeit, eine Verbindung zu SMTP-, POP3-, IMAP- und Exchange-Server herzustellen.
- Funktioniert mit Thunderbird, Zimbra und IBM Notes.

### E-Mail-Formate lesen und schreiben

**Microsoft Outlook®:** MSG, PST, OST, OFT\
**E-Mail:** EML, EMLX, MBOX\
**Andere:** ICS, VCF, HTML, MHTML

### E-Mail-Formate lesen

**Mac Outlook:** OLM

### Unterstützte Umgebungen

- **Microsoft Windows®:** Windows-Desktop und -Server (x86, x64)
- **macOS:** Mac OS X
- **Linux:** Ubuntu, OpenSUSE, CentOS und andere
- **Java-Versionen:** `J2SE 7.0 (1.7)`, `J2SE 8.0 (1.8)`

### Loslegen

Aspose hostet alle Java-APIs unter [Aspose Repository](https://repository.aspose.com/webapp/#/artifacts/browse/tree/General/repo/com/aspose/aspose-email). Sie können Aspose.BarCode for Java API mit einfachen Konfigurationen direkt in Ihren Maven-Projekten verwenden. Detaillierte Anweisungen finden Sie auf der Dokumentationsseite [Installing Aspose.Email for Java from Maven Repository](https://docs.aspose.com/email/java/installation/).

## 为什么要下载 Aspose.Note for Java？

Aspose.Note for Java 是一个类库，它使应用程序无需在服务器上安装即可以编程方式与 Microsoft Office OneNote® 交互。它是 MS OneNote® 对象模型的纯替代品，提供更好的性能和易用性来管理 OneNote® 文档。 Aspose.Note for Java API 提供读取、转换、创建和编辑/操作 Microsoft OneNote® 文件格式内容的功能。

## Microsoft OneNote® 文件处理

- 加载、保存和转换 OneNote® 文档。
- 在 OneNote® 中生成根和子级别页面。
- 获取页面修订并回滚到以前的版本。
- 从 OneNote® 文档中提取图像。
- 从 OneNote® 文档的指定页面中提取或替换文本。
- 在 OneNote® 文档中创建一个带有锁定列的表格。
- 将文件附加到 OneNote® 文档。
- 创建、保存、阅读、转换 OneNote® Notebook。

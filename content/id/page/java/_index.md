### Mengapa mengunduh Aspose.Page untuk Java?

Aspose.Page for Java menawarkan untuk membuat & memanipulasi dokumen XPS. Ini juga menyediakan fungsionalitas untuk mengonversi dokumen XPS, PS & EPS ke format gambar & PDF.

### Memanipulasi Dokumen XPS melalui Java

- Membuat dan mengedit dokumen XPS.
- Menambah atau menghapus halaman dokumen XPS.
- Simpan dokumen XPS sebagai gambar raster & PDF.
- Bekerja dengan elemen kanvas, jalur, dan mesin terbang.
- Buat bentuk grafik vektor (elemen Path) menggunakan satu set primitif (busur elips, segmen kurva Bezier dan segmen garis lurus).
- Buat string teks (elemen Glyphs).
- Elemen grup (elemen kanvas) untuk memanipulasi grup secara keseluruhan.
- Memanipulasi tampilan grafik dan string teks.
- Gunakan kuas dari berbagai jenis termasuk kuas warna solid, kuas gambar, kuas visual, dan lainnya.
- Tentukan warna dalam ruang warna yang berbeda termasuk sRGB, scRGB, dan ruang apa pun berdasarkan profil ICC.
- Bekerja dengan banyak dokumen di dalam dokumen XPS.
- Simpan tiket cetak dalam dokumen XPS yang terbuka jika ada.
- Tambahkan tiket cetak default ke dokumen XPS baru atau yang dibuka jika tidak ada tiket di dalamnya.
- Memanipulasi hyperlink yang terkait dengan elemen XPS.
- Memanipulasi, menyimpan, dan mengonversi item garis besar XPS.

### Konversi File PostScript melalui Java

- Mendukung level bahasa PostScript 1-3 dengan pengecualian jenis font: Type2 (CFF), Type14 (Chameleon), Types 9, 10, 11, 32 (CID-Keyed)
- Simpan PS & EPS sebagai PDF & gambar.

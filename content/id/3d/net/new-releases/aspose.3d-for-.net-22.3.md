# Peningkatan Mesh Besar dalam Ekspor U3D/PDF

Meningkatkan mesh besar dalam mengekspor file U3D / PDF.

# Penggabungan Adegan 3D yang Disederhanakan

Menyederhanakan fungsi untuk menggabungkan adegan. Metode baru ini memungkinkan untuk menggabungkan semuanya dari node lain ke node saat ini.

```csharp
var scene1 = Scene.FromFile("file1");
var scene2 = Scene.FromFile("file2");
scene1.RootNode.Merge(scene2.RootNode);
scene1.Save("output.fbx", FileFormat.FBX7700Binary);
```

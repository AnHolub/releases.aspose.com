## Mengapa mengunduh Aspose.OCR untuk Java?

Aspose.OCR untuk Java adalah API pengenalan tanda karakter dan optik yang memungkinkan pengembang untuk menambahkan fungsionalitas OCR dalam aplikasi mereka. Ini menyediakan satu set kelas sederhana untuk mengontrol pengenalan karakter.

## Konversi Gambar ke Teks melalui Java

- Baca karakter dari gambar.
- Dukungan untuk format file gambar JPG, PNG, GIF, BMP dan TIFF untuk OCR.
- Dukungan untuk bahasa Inggris, Prancis, Spanyol, dan Portugis.
- Baca font populer termasuk Arial, Times New Roman, Courier New, Verdana, Tahoma dan Calibri.
- Dukungan untuk gaya font reguler, tebal, dan miring.
- Pindai seluruh gambar atau bagian mana pun dari gambar.
- Pindai gambar yang diputar.
- Dapat menerapkan filter penghilang derau yang berbeda sebelum pengenalan gambar.
- Ekstrak elemen bertanda optik dari gambar yang dipindai.
- Dukungan untuk berbagai elemen bertanda optik seperti Kotak Centang, Kotak Pilihan, Kotak, Kode Batang & OCR.

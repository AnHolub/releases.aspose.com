## Mengapa mengunduh Aspose.BarCode untuk Java?

[Aspose.BarCode for Java](https://products.aspose.com/barcode/java) adalah komponen pengenalan dan pembuatan kode batang yang kuat dan andal, yang ditulis dalam Java, memungkinkan pengembang untuk dengan cepat dan mudah menambahkan fungsi pengenalan dan pembuatan kode batang ke aplikasi Java mereka.

## Fitur API Kode Batang

- Tentukan berbagai jenis simbologi.
- Atur teks Kode barcode, tampilan dan properti lainnya.
- Dukungan Checksum dan Data Tambahan.
- Cetak gambar barcode.

### Simbologi Kode Batang

- **Numerik Saja:** EAN13, EAN8, UPCA, UPCE, BooklandEAN, Interleaved2of5, Standard2of5, MSI, Code11, Codabar, Postnet, Planet, EAN14(SCC14), SSCC18, ITF14, Leticode, OPC
- **Alpha-Numeric:** Code128, EAN128, Code39 Extended, Code39 Standard, Code93, Extended, Code93 Standard, Matrix 2 of 5, PZN, Deutsche Post Identcode, VIN
- **Simbol 2D:** PDF417, DataMatrix, QR, Swiss QR (QR Bill)
- **Dukungan Encoding Saja:** Australia Post, Aztec

### Membaca & Menulis Label Barcode

**Gambar:** JPEG, TIFF, PNG, BMP, GIF, EXIF, EMF, SVG

### Simpan Barcode Sebagai

**Gambar:** JPEG, TIFF, PNG, BMP, GIF, EXIF, EMF, SVG

### Lingkungan yang Didukung

- **OS apa pun yang mendukung Java:**
- **Java Runtime:** `JRE 1.8` atau lebih tinggi
- **Pengembangan Java:** `J2SE 8.0 (1.8)`, atau lebih tinggi.

### Зачем скачивать Aspose.Page для Java?

Aspose.Page for Java позволяет создавать документы XPS и управлять ими. Он также предоставляет функции для преобразования документов XPS, PS и EPS в форматы изображений и PDF.

### Управление XPS-документами через Java

- Создание и редактирование XPS-документа.
- Добавление или удаление страниц документа XPS.
- Сохраняйте документы XPS как растровые изображения и PDF.
- Работа с холстами, дорожками и элементами глифов.
- Создание фигур векторной графики (элемент Path) с использованием набора примитивов (эллиптические дуги, сегменты кривой Безье и сегменты прямых).
- Создание текстовых строк (элемент Glyphs).
- Элементы группы (элемент Canvas) для управления группой в целом.
- Управление внешним видом графических и текстовых строк.
- Используйте кисти разных типов, включая сплошную цветную кисть, кисть изображения, визуальную кисть и другие.
- Укажите цвета в различных цветовых пространствах, включая sRGB, scRGB и любое пространство на основе профиля ICC.
- Работа с несколькими документами внутри документа XPS.
- Сохраняйте билеты на печать в открытом документе XPS, если таковые имеются.
- Добавляйте квитанции на печать по умолчанию в новый XPS-документ или открытый документ, если в нем нет квитанций.
- Управление гиперссылками, связанными с элементами XPS.
- Управление, сохранение и преобразование элементов структуры XPS.

### Преобразование файлов PostScript через Java

- Поддерживает уровни языка PostScript 1–3, за исключением типов шрифтов: Type2 (CFF), Type14 (Chameleon), Type 9, 10, 11, 32 (CID-Keyed).
- Сохраняйте PS и EPS как PDF и изображения.

### Поддерживаемые среды

- **Microsoft Windows®:** Windows Desktop & Server (x86, x64)
- **macOS:** Mac OS X
- **Linux:** Ubuntu, OpenSUSE, CentOS и др.
- **Версии Java:** `J2SE 8.0 (1.8)` или выше

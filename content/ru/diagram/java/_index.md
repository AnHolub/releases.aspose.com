## Зачем скачивать Aspose.Diagram для Java?

Aspose.Diagram for Java API — это решение для работы с файлами Microsoft Visio®. Он позволяет приложениям Java читать, записывать, экспортировать и обрабатывать диаграммы Microsoft Visio®, поддерживая при этом все объекты и свойства Visio®, включая фигуры, страницы, изображения, образцы форм, наборы элементов, текст, слои, верхний и нижний колонтитулы, определяемые пользователем ячейки, гиперссылки, защита файлов, геометрия, текстовые поля и комментарии.

Aspose.Diagram для Java также можно использовать для защиты рисунков Microsoft Visio® путем блокировки фона, трафаретов (мастер), а также форм и стилей, чтобы избежать случайных изменений.

## Функции обработки файлов Visio®

- Загружайте, создавайте, сохраняйте и конвертируйте диаграммы Microsoft Visio®.
- Создайте мастер из набросков или добавьте из трафарета форм.
- Создание, компоновка и автоматическая подгонка фигур, а также получение информации о коннекторах Visio® и шрифтах.
- Изменить раздел геометрии формы соединителя.
- Вставьте элемент управления ActiveX в диаграмму Visio®.
- Печать на определенный принтер или установка принтера и имени документа.

### Чтение и запись форматов Visio®

**Microsoft Visio®:** VSDX, VSX, VTX, VDX, VSSX, VSTX, VSDM, VSSM, VSTM

### Сохранить диаграммы Visio® как

**Фиксированный макет:** PDF, XPS\
**Изображения:** JPEG, PNG, BMP, TIFF, SVG, EMF\
**Интернет:** HTML\
**Другое:** XAML, SWF

### Чтение форматов Visio®

**Microsoft Visio®:** VDW, VSD, VSS, VST

### Поддерживаемые среды

- **Microsoft Windows®:** Windows Desktop & Server (x86, x64)
- **macOS:** Mac OS X
- **Linux:** Ubuntu, OpenSUSE, CentOS и др.
- **Версии Java:** `J2SE 7.0 (1.7)`, `J2SE 8.0 (1.8)`

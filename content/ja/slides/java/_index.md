## Aspose.Slides for Java をダウンロードする理由

[Aspose.Slides](https://products.aspose.com/slides/java) for Java は、Java アプリ内から Microsoft PowerPoint® プレゼンテーションを操作するためのオンプレミス API です。開発者は、スライド、表、テキスト、グラフ、図形、画像、SmartArt 図などのさまざまなドキュメント要素を操作する機能を使用して、Java アプリケーションで PowerPoint® プレゼンテーションの読み取り、書き込み、変換、および操作を行うことができます。

## Java による PowerPoint® ファイル操作

- プレゼンテーションを読み込んで他の形式に変換します。
- プレゼンテーションをゼロから作成します。
- 直感的なオブジェクト モデルを介してすべてのプレゼンテーション要素を操作します。
- グラフを作成または埋め込む。
- 形状を作成または操作します。

Aspose.Slides for Java の詳細については、[product overview](https://docs.aspose.com/slides/java/product-overview/) をチェックしてください。

### 読み書きプレゼンテーション

**Microsoft PowerPoint®:** PPT、PPTX、PPS、POT、PPSX、PPTM、PPSM、POTX、POTM\
**OpenOffice:** ODP、OTP

### プレゼンテーションに名前を付けて保存

**固定レイアウト:** PDF、PDF/A、XPS\
**画像:** JPEG、PNG、BMP、TIFF、GIF、SVG\
**ウェブ:** HTML、SWF

### サポートされている環境

- **Microsoft Windows®:** Windows デスクトップおよびサーバー (x86、x64)
- **macOS:** Mac OS X
- **Linux:** Ubuntu、OpenSUSE、CentOS など
- **Java バージョン:** `J2SE 6.0 (1.6)` 以降
